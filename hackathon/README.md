🚗 Car Wash Data Analytics & Dashboard Project

🚀 Overview

This project analyzes car wash business data to understand sales trends, customer behavior, and service performance using Python.

🎯 Problem Statement

Car wash businesses struggle to track performance, identify popular services, and understand seasonal demand.

💡 Solution

Analyzed multiple datasets to:

- Track monthly sales trends
- Identify top services
- Analyze customer churn
- Understand seasonality patterns

🧠 Tools Used

- Python (Pandas, Matplotlib)

📊 Key Insights

- Found peak sales months
- Identified high-demand services
- Observed seasonal trends
- Analyzed customer retention

📁 Project Files

- analysis.py → data analysis
- visualization.py → charts
- main.py → execution
- CSV files → datasets

📌 Conclusion

This project helps in making data-driven decisions to improve business performance.
